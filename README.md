# Student Career Outcomes & Engagement Dashboard

## 📌 Project Overview
This project analyzes student career outcomes, engagement levels, and hiring trends using Excel, simulating how a university career center can use data to support data-driven decision-making.
---

## 🎯 Objectives
- Analyze employment outcomes of students  
- Understand the impact of career engagement on success  
- Identify top hiring industries and salary trends  
- Build an interactive dashboard for easy insights  

---

## 🛠️ Tools Used
- Microsoft Excel (Data Cleaning, INDEX-MATCH, Pivot Tables, Dashboard)
---

## 📊 Data Structure
The dataset was designed using a multi-sheet structure:

- **Students** → Student demographics and academic details  
- **Engagement** → Career center activity (workshops, resume reviews, etc.)  
- **Outcomes** → Internship, employment status, salary, and industry  
- **Master_Data** → Integrated dataset using INDEX-MATCH  

This structure demonstrates real-world data modeling and integration techniques.

---

## ⚙️ Key Features
- Built a centralized master dataset by integrating multiple data sources using lookup functions and **INDEX-MATCH **, simulating real-world data workflows
- Cleaned and validated data using Excel formulas and logic checks  
- Created **PIVOT TABLES** for structured analysis  
- Designed an interactive Excel dashboard with charts and KPIs  
- Implemented **dynamic filtering (slicers)** to view all metrics by major  
  

---

## 📈 Dashboard Highlights
The dashboard provides insights into:

- Employment distribution (Employed, Seeking, Further Study)  
- Outcomes by major  
- Top hiring industries  
- Salary trends across majors  
- Relationship between student engagement and employment outcomes  

---

## 💡 Key Insights
- Higher student engagement is associated with better employment outcomes  
- Technology and Finance are among the top hiring industries  
- Students with internships are more likely to secure full-time roles  
- Certain majors show consistently higher employment rates
